# 🍔 Ravenous

Ravenous is a responsive restaurant discovery application built with **React**. It leverages the **Yelp Fusion API** to provide users with a seamless way to search for local businesses, view ratings, and sort results based on their preferences.

## 🚀 Key Features
* **Live Data:** Fetches real-time business information using the Yelp API.
* **Search Functionality:** Filter by search term (e.g., "Pizza") and location.
* **Dynamic Sorting:** Sort results by "Best Match," "Highest Rated," or "Most Reviewed."
* **Clean UI:** A modern, mobile-responsive layout built with CSS Modules.

## 🛠️ Technology Stack
* **Frontend:** React.js
* **Styling:** CSS Modules
* **Data Fetching:** Fetch API / Yelp Fusion API
* **Environment:** Node.js

## ⚙️ Installation & Setup
1. **Clone the repository:**
   ```bash
   git clone https://github.com/benpaymaster/ravenous.git
   ```
2. **Install dependencies:**
   ```bash
   npm install
   ```
   npm start
   ```
